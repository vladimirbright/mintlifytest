# Overview

Truv’s **Verification of Employment History (VOE)** product allows you to access all users’ employment information from authenticated accounts. It helps you verify anyone’s employment quickly and makes the decision process easier.

Truv gathers data from a trusted source that provides employment history. Employment history is helpful for background screening or employment verification. This data includes user identity, employer, and employment status.

With VOE, give background screeners, loan officers, banks, fintech, and other organizationss an exceptional experience. Users can complete verification instantly instead of waiting for a few days.

## Use cases for employment history

These points are example scenarios for verifying employment history.

* Pre-employment screening
* Loan origination to confirm employment before closure
