# Verification data

The data below refers to insurance information verification.

| Name                    | Type   | Description                                                                  |
| ----------------------- | ------ | ---------------------------------------------------------------------------- |
| id                      | string | Unique ID of the report                                                      |
| status                  | string | Task status from the [Connection Lifecycle](/reference/connection-lifecycle) |
| finished\_at            | string | Time when report was finished                                                |
| access\_token           | string | Payroll provider Link access\_token                                          |
| tracking\_info          | string | String value provided when generating `bridge_token`                         |
| [insurance](#insurance) | object | Insurance policy information                                                 |

## Insurance

This table covers high-level insurance information from the insurance provider records.

| Name                                 | Type  | Description                    |
| ------------------------------------ | ----- | ------------------------------ |
| [policies](#policies)                | array | List of policy objects         |
| [drivers](#drivers)                  | array | List of vehicle objects        |
